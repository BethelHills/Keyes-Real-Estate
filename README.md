# Keyes Real Estate Website

This is the official website project for Keyes Real Estate.
Built and maintained by BCodeStack.
The platform helps users buy, sell, and rent properties through a clean interface, smart search system, and responsive design.

## Project Overview

Keyes Real Estate is designed to:
- Display verified property listings
- Provide advanced property search and filters
- Allow direct contact with agents
- Deliver fast performance across all devices
- Maintain a professional and modern user experience

The system is built to be scalable and easy to maintain.

## Tech Stack

This project is built with:
- Vite
- TypeScript
- React
- Tailwind CSS
- Shadcn UI

Structured for performance and clean code management.

## Local Development Setup

Make sure Node.js and npm are installed.
Then run:

```bash
# Clone the repository
git clone https://github.com/BethelHills/Keyes-Real-Estate.git

# Enter the project directory
cd Keyes-Real-Estate

# Install dependencies
npm install

# Start the development server
npm run dev
```

The project will run locally with live reload enabled.

## Project Structure

Main directories:
- `src/` → Application logic and components
- `components/` → Reusable UI components
- `pages/` → Page layout structure
- `public/` → Static files and assets

The structure is optimized for scalability and clean development workflow.

## Deployment

To deploy via Lovable:
1. Open the project inside Lovable
2. Click Share
3. Click Publish

For manual deployment, build using:

```bash
npm run build
```

Deploy the generated `dist` folder to your hosting provider.

## Custom Domain Setup

To connect a domain: https://keyes-real-estate-nr3x.vercel.app

Project → Settings → Domains → Connect Domain

Follow domain configuration instructions inside the dashboard.
